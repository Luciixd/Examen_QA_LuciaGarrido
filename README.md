# Examen de Casos de Prueba - QA Intermedio

## Autor
**Lucia Gabriela Garrido Morales**

## Descripción
Este repositorio contiene el desarrollo completo del **Examen de Casos de Prueba - QA (Nivel Intermedio)**.  
Incluye la documentación de casos de prueba, reporte de defectos y evidencias visuales de ejecución.  
Se trabajaron tres escenarios basados en plataformas reales:

1. **Saga Ripley** – Caso de regresión del módulo de pagos con cupón de descuento.  
2. **YouTube** – Proceso de subida de videos con manejo de errores.  
3. **Facebook** – Pruebas exploratorias de comentarios con texto largo, emojis y menciones.


## Contenido de los Documentos

### **Saga_Ripley.docx**
Contiene los tres flujos de prueba:
- Flujo feliz (cupón válido)
- Flujo alterno (cupón expirado)
- Flujo de error (falla en pasarela de pago)  
Cada flujo incluye capturas de pantalla dentro del documento.


### **YouTube.docx**
Casos de prueba para subida de video:
- Flujo feliz (video válido subido correctamente)
- Flujo alterno (error por conexión inestable)
- Flujo de error (archivo no compatible)  
Las evidencias visuales están integradas en el documento Word.

###  **Facebook.docx**
Casos exploratorios:
- Flujo feliz (comentario largo con emojis y menciones)
- Flujo alterno (comentario excede límite)
- Flujo de error (fallo al procesar comentario o conexión)  
Cada caso incluye imágenes insertadas como evidencia.



##  Reporte de Defectos
En cada documento se incluye un bloque de defectos con:
- **Bug**  
- **Incidencia**  
- **Error**  
Cada uno documentado con título, descripción, resultado actual/esperado, severidad y prioridad.
